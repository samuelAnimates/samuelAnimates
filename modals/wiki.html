WHEELCHAIR TRAVEL WIKI

2017

Wheelchair travel wiki + mobile app

MERN stack application in-progress

Key Question:

How can we present wheelchair accessibility info in a flexible, usable contributor-friendly format?

Current Status:

  • Github: [https://github.com/smendez92/wheelchairWiki]
  • Heroku: [Coming Soon]

Areas of Exploration:

  • How can we design a flexible database system that will leave room to merge/grow with other datasets?
  • How do we create an effective minimum standard of detail that will help lower the barrier for contributors while still providing usable info?
  • How do we begin such a project with web accessibility in mind?

Background:

Starting with [previous work] as an inspiration, we want to make a searchable, flexible, collective database to store usable information about wheelchair accessible travel. Visit us on github to check out our work-in-progress.

Process:

In-progress. Prototype to be presented by November 2017.
