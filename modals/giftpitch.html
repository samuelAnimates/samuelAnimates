GIFTPITCH

2017

Web app

HTML, CSS, JavaScript.

Synopsis

First class project for Northwestern Coding Boot Camp: a web app that lets users take a personality quiz about a loved one to receive a tailored set of gift options for them from Amazon, Etsy, and Ebay.

Role

Backend developer + content writer.

Goal

Work together in a team to create an application with a polished frontend, api calls, and a technology new to us at the time. We decided to make a gift-suggesting app using API calls to Ebay, Etsy, and Amazon, dabbling into frontend frameworks to help us display the results in image carousels.


My Role

Developed/coded personaltiy quiz logic, co-authored personality quiz text, co-authored gift result text, and tailored API call parameters to return related but distinct gift options from each store.


Process
